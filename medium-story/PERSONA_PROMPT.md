# medium-story — Persona Configuration

> **CUSTOMISE THIS FILE** — edit the persona section to match your voice.
> See `templates/persona-template.md` in the Hermes-Skills repo for a full template.

```yaml
persona:
  name: {{YOUR_NAME}}
  title: {{YOUR_TITLE}}
  voice: "British English, direct, practitioner, dry wit"
  style_guide: "templates/persona-template.md"
```

## Quick Start

1. Copy `templates/persona-template.md` to your repo
2. Fill in the `{{YOUR_NAME}}`, `{{YOUR_TITLE}}`, and `{{YOUR_BIO}}` fields
3. Add your specific writing patterns under the "CUSTOMISE THIS SECTION" heading
4. Point the `PERSONA_FILE` variable in `SKILL.md` to this file
